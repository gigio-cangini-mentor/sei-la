# Template: Anúncio e Criativo — Framework Gancho+Corpo+CTA

> **Base:** Pedro Sobral — "Anúncios Impossíveis de Ignorar" (Gancho + Corpo + CTA)
> **Voz do criativo:** Ícaro de Carvalho — PPP aplicado ao copy do anúncio
> **Orgânico vs. Pago:** estrutura compartilhada, execução adaptada (Juliana Gomes para orgânico)
>
> **Princípio central:** "O gancho é o critério de vida ou morte do criativo. Se não para o scroll, nada mais importa." — Sobral
> **Filtro de voz:** o criativo não precisa vender para todo mundo — precisa parar a pessoa certa

---

## A Lógica por Trás da Estrutura

```
GANCHO (0–3 segundos / primeira linha de texto)
→ Para o scroll. Filtra a audiência. Promete algo que vale o tempo.

CORPO (o desenvolvimento)
→ Sustenta o gancho. Contexto + mecanismo + prova emocional ou racional.

CTA (call to action)
→ Uma instrução. Uma ação. Sem ambiguidade.
```

**Regra de ouro do Sobral:** o gancho não é para todo mundo — é para a pessoa certa. Um gancho que filtra audiência errada reduz custo por resultado.

**Regra de ouro do Ícaro:** o criativo não esconde a venda. Transparência sobre intenção comercial não reduz conversão — aumenta confiança.

## Placeholders

| Placeholder | O que preencher |
|-------------|-----------------|
| `{{GANCHO}}` | Primeira frase ou conceito visual — para o scroll em 3 segundos |
| `{{PROBLEMA}}` | Dor específica e reconhecível da audiência-alvo |
| `{{SOLUCAO}}` | O que o produto/serviço entrega — resultado, não feature |
| `{{PROVA}}` | Número, depoimento ou resultado específico |
| `{{CTA}}` | Verbo de ação único: "clique", "arraste", "acesse", "me chame" |
| `{{URL}}` | Link de destino (página de vendas, WhatsApp, bio, etc.) |
| `{{PRODUTO}}` | Nome do produto ou serviço |
| `{{AUDIENCIA}}` | Quem você está chamando explicitamente |

## ESTRUTURA BASE (Todos os Formatos)

```
[GANCHO]
Para o scroll. Promessa, provocação, surpresa ou identificação imediata.
Máximo: 1 a 2 frases / 3 segundos de vídeo.

[CORPO]
Contexto da dor → por que acontece → mecanismo da solução → prova.
Extensão varia por plataforma (ver abaixo).

[CTA]
Uma ação. Um link. Sem opções paralelas.
```

## VARIAÇÕES POR PLATAFORMA

### Facebook / Instagram — Feed (Imagem ou Vídeo Estático)

**Características:** mais espaço para copy; leitor em modo "exploração"; aceita texto mais longo

```
GANCHO (primeiras 2 linhas — o que aparece sem clicar "ver mais"):
{{GANCHO}}

CORPO:
{{PROBLEMA}}.

[Por que isso acontece — 1 a 2 frases explicando o mecanismo]

{{SOLUCAO}}.

[Prova ou resultado específico:]
{{PROVA}}

CTA:
{{CTA}} → {{URL}}
```

**Regras para Feed:**
- As duas primeiras linhas são o gancho textual — antes do "ver mais"
- Texto longo funciona se o gancho segurar; texto curto funciona sempre
- Imagem é o gancho visual — deve parar o scroll antes do texto ser lido
- Nunca começar com o nome da marca ou do produto

---

### Instagram / Facebook — Stories e Reels (Vídeo Vertical)

**Características:** atenção altíssima e brevíssima; sem texto longo; gancho visual é o criativo

```
GANCHO (0–3 segundos — imagem ou fala de abertura):
{{GANCHO}}
[Visual: o que aparece na tela antes de qualquer fala]

CORPO (3–25 segundos):
[Dor em uma frase]
[Mecanismo ou transformação em 2 a 3 frases]
[Prova: número ou resultado em uma frase]

CTA (últimos 3–5 segundos):
{{CTA}} — {{URL}} ou "arrasta pra cima" / "link na bio"
```

**Regras para Stories/Reels:**
- Gancho visual precisa funcionar sem som (50%+ assiste sem áudio)
- Legenda/transcrição automática é obrigatória para quem não ativa som
- CTA no vídeo E na legenda — não confie só em um
- Formato Recipe (lista numerada) tem maior retenção em Reels — use quando possível

---

### Google Search (Texto)

**Características:** intenção de busca já existe; gancho é relevância para o termo buscado

```
HEADLINE 1 (30 chars): [Palavra-chave principal + proposta]
HEADLINE 2 (30 chars): [Diferencial ou resultado]
HEADLINE 3 (30 chars): [CTA ou urgência]

DESCRIÇÃO 1 (90 chars):
{{PROBLEMA}}. {{SOLUCAO}} — {{PROVA}}.

DESCRIÇÃO 2 (90 chars):
[Benefício secundário ou objeção resolvida] + {{CTA}}.

URL de destino: {{URL}}
```

**Regras para Google Search:**
- Headline 1 deve espelhar o termo de busca — relevância antes de criatividade
- Nunca usar CAPS LOCK nas headlines
- Cada headline deve funcionar sozinha — elas rotacionam e combinam automaticamente
- Descrição é onde a persuasão acontece; headline é onde a relevância é confirmada

## ADAPTAÇÃO: ORGÂNICO (Framework Juliana Gomes) vs. PAGO (Framework Sobral)

### Orgânico — Conteúdo que Converte no Instagram

> Quando o criativo não tem verba por trás, ele precisa trabalhar pelas 3 Funções do Conteúdo (Juliana Gomes). Defina a função antes de criar: ganhar seguidor, engajar ou vender.

**Função 3 — Vender (orgânico):**

```
GANCHO (hook dos primeiros 3 segundos / primeira linha da legenda):
{{GANCHO}}
[Para o scroll. Cria curiosidade ou identificação de dor — não mencione o produto ainda]

CORPO (aquecimento antes da oferta):
[Conteúdo educativo ou prova social relacionado ao produto]
[2 a 3 dias antes: Stories de aquecimento sobre o tema]
[No post de venda: resultado de aluno ou case específico]
{{PROVA}}

CTA (clara, explícita, única):
"{{CTA}} — link na bio"
ou "Me chama no DM com {{PALAVRA-CHAVE}}"
ou "Arrasta para acessar"
```

**Regras para orgânico:**
- Nunca ofereça sem aquecer — "namoro antes do casamento" (2 a 3 dias de conteúdo antes)
- Prova social é conteúdo — prints e resultados de alunos vendem mais que copy elaborado
- CTA explícita é obrigatória — "quem não oferece, não vende" (Juliana Gomes)
- Formato Recipe (lista, passo a passo) tem maior compartilhamento e alcance no orgânico

### Pago — Anúncio com Verba (Sobral)

> Com verba, o gancho é testável. Rode variações de gancho antes de escalar orçamento.

**Protocolo de teste de gancho (Sobral):**
1. Crie 3 versões do mesmo criativo com ganchos diferentes
2. Mesmo público, mesmo orçamento, mesmo destino
3. Meça CTR nos primeiros 3 dias
4. O gancho com maior CTR recebe a verba; os outros, pausa
5. Nunca escale orçamento antes de validar o gancho

## 3 EXEMPLOS PREENCHIDOS

### Exemplo 1 — Infoproduto (Curso de Copywriting)

**Plataforma:** Instagram Feed
**Audiência:** empreendedores digitais, freelancers, profissionais de marketing

```
GANCHO:
Você não tem problema de vendas. Você tem problema de copy.

CORPO:
A maioria das pessoas acha que o produto é bom mas a venda não acontece.
Na maior parte dos casos, o produto é bom mesmo. O problema está em como
você descreve o resultado que ele entrega.

Quando você aprende a escrever o que o cliente já está sentindo — antes
de ele saber que você existe —, a resistência à compra desaparece.

Isso é o que eu ensino em 8 semanas no [Nome do Curso]:
como transformar o que você faz em copy que converte.

Mais de 3.200 alunos aplicaram esse método.

CTA:
Acesse a página e veja o que está incluído → [URL]
```

**Por que funciona:** gancho derruba a crença mais comum ("preciso melhorar o produto") e imediatamente redireciona para o problema real. O "antes de ele saber que você existe" cria curiosidade sobre o mecanismo.

---

### Exemplo 2 — Serviço (Consultoria de Posicionamento)

**Plataforma:** Stories/Reels vertical
**Audiência:** profissionais liberais cobrando abaixo do potencial

```
GANCHO (0–3 seg, fala ou texto na tela):
"Se você está lotado mas não está satisfeito com o que recebe, não é
problema de preço — é problema de proposta."

CORPO (3–25 seg):
Eu já atendi mais de 200 profissionais que tinham agenda cheia e
faturamento baixo.

O padrão é sempre o mesmo: eles cobram pela hora, não pelo resultado.

Quando eles aprendem a vender o resultado — não o tempo —, o preço
que conseguem cobrar muda completamente.

Em média, os clientes que trabalham comigo dobram o ticket em 60 dias.

CTA (últimos 5 seg):
Me chama no DM com "proposta" que eu te explico como funciona.
```

**Por que funciona:** gancho de reframing (problema de proposta, não de preço) filtra exatamente quem tem o problema. Número concreto ("dobram o ticket em 60 dias") sustenta o corpo sem precisar de copy longo.

---

### Exemplo 3 — E-commerce (Suplemento de Performance)

**Plataforma:** Facebook/Instagram Feed
**Audiência:** homens 25–40 ativos que treinam com frequência

```
GANCHO:
Treinar todo dia e não evoluir não é falta de dedicação. É falta de
recuperação.

CORPO:
A maioria das pessoas que trava no treino está dormindo mal,
comendo tarde e com cortisol alto.

O resultado? Você treina igual mas o músculo não cresce.

[Nome do produto] foi formulado para atacar exatamente isso:
magnésio quelato + L-teanina + adaptógenos para recuperação real.

Não é pré-treino. É o que seu corpo precisa depois.

Mais de 8.000 avaliações. Média 4.8 estrelas.

CTA:
Compre agora com frete grátis → [URL]
```

**Por que funciona:** gancho desmonta a culpa ("não é falta de dedicação") e cria abertura para a solução. "Não é pré-treino" diferencia o produto da categoria mais saturada. Prova social com número concreto fecha o argumento.

## Anti-Padrões — O Que Nunca Fazer

| Anti-padrão | Por que mata o anúncio |
|-------------|------------------------|
| Começar com o nome da marca | A marca não para o scroll — a dor ou a promessa param |
| Gancho genérico ("Você quer mudar sua vida?") | Fala para todo mundo = não fala para ninguém |
| Múltiplos CTAs ("Clique aqui, ou acesse nosso site, ou nos chame") | Paralisia de escolha; zero ação é o resultado |
| Proposta racional no gancho ("Curso com 12 módulos") | Racional não para scroll; emoção para |
| CAPS LOCK excessivo | Parece spam; reduz CTR em campanhas pagas |
| Esconder que é anúncio | No pago, o Facebook já marca; no orgânico, a transparência aumenta confiança |
| Gancho longo demais | Mais de 2 frases no gancho não é gancho — é corpo |
| CTA sem link ou instrução clara | "Saiba mais" sem URL não converte |

## Checklist de Revisão Antes de Publicar

**Para todos os formatos:**
- [ ] O gancho para o scroll sem precisar ler o resto?
- [ ] A dor está descrita com palavras que a audiência usaria (não jargão)?
- [ ] Existe um único CTA com uma instrução clara?
- [ ] A venda está explícita — não disfarçada de conteúdo?
- [ ] Tem prova específica (número, resultado ou depoimento real)?

**Para pago (Sobral):**
- [ ] Criou pelo menos 3 variações de gancho para testar?
- [ ] Definiu a métrica de decisão antes de rodar (CTR mínimo)?
- [ ] O destino (LP ou página) está alinhado com o que o gancho prometeu?

**Para orgânico (Juliana Gomes):**
- [ ] A função do post está definida (ganhar seguidor / engajar / vender)?
- [ ] Se é post de venda, houve aquecimento nos 2 a 3 dias anteriores?
- [ ] O hook retém até o final (taxa de retenção prevista acima de 50%)?
