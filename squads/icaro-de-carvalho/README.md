## Frameworks Principais

### Ícaro — Produto, Proposta e Personalidade (Framework Central)
Todo objeto comercial = Produto (racional) + Proposta (emocional) + Personalidade (voz única).
No Brasil, Proposta > Produto.

### Ícaro — Estratégia Tripwire
Conteúdo gratuito massivo → única oferta forte, alto ticket, escassez real.

### Ícaro — Sprint (Tiro Curto)
4 axiomas: pessoas compram → de alguém → de quem confiam → confiança vem do conteúdo.

### Sobral — Método Subido (Tráfego Pago)
Framework completo de gestão de tráfego: Planejamento → Estruturação → Distribuição → Resultados → Otimização.

### Sobral — CTR Subido
Gancho 85% / Corpo 10% / CTA 5% — a proporção de esforço num anúncio.

### Kiso — Unbound Marketing (5 Passos)
Diagnóstico → Objetivos → Estratégia → Tática → Controle. Integra inbound + outbound + influência + indicação.

### Kiso — Jornada Bumerangue
Descoberta → Consideração → Conversão → Experiência Própria → Experiência Compartilhada (ciclo virtuoso).

### Hiller — Branding como Ativo Estratégico
Marca não é logo — é promessa percebida. Relevância > Reconhecimento.

### Hiller — Brand Persona (4 Dimensões)
Arquétipos + Tom de Voz + Valores + Propósito = Marca com personalidade.

## Diferenciação

### vs copywriting-squad
O **copywriting-squad** usa copywriters internacionais lendários (Hopkins, Halbert, Schwartz). Este squad usa **frameworks brasileiros proprietários** do Ícaro de Carvalho (PPP, Tripwire, Sprint), desenvolvidos para o mercado de infoprodutos brasileiro. Se o copy precisa soar americano e clássico, use o copywriting-squad. Se precisa soar como o Novo Mercado, use este.

### vs leandro-ladeira
O **squad Leandro Ladeira** foca em vendas diretas e conversão extrema. Este squad cobre o **ciclo completo**: branding (Hiller) → conteúdo orgânico (Juliana) → copy e lançamento (Ícaro) → tráfego pago (Sobral) → analytics (Kiso). É uma operação de marketing de ponta a ponta.
