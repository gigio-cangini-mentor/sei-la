# Como Fazer Testes no Tráfego Pago

## Definição Central

**Frase exata:** "Testar é, basicamente, investir qualquer valor em uma campanha e analisar os resultados."

**Princípio absoluto:** "Cada centavo investido em tráfego pago é um teste." Isso se aplica se você está rodando testes comparativos ou campanhas de variável única.

## Estrutura de Campanha para Testes

### Nível de Campanha
- Objetivo (impacto alto ⭐)
- Manual ou Advantage
- CBO/ABO (alocação de orçamento) ⭐
- Orçamento diário/total ⭐
- Programação
- Estratégia de lance

### Nível do Conjunto de Anúncios
- Evento de conversão ⭐
- Público-alvo (original) ⭐
- Tipo de público (personalizado, localização, idade, gênero)
- Advantage+ com sugestões

### Nível do Anúncio
- Formato (flexível, imagem, vídeo, carrossel) ⭐
- Mídia (imagem/vídeo) ⭐
- Destino/CTA ⭐

## Cinco Metodologias de Teste

### 1. Testar Enquanto Roda
Monitorar e ajustar com essas frequências:
- Orçamento: a cada 2 dias
- Públicos: a cada 4 dias
- Criativos: a cada 2-3 dias
- Estrutura de campanha: a cada 7 dias

### 2. Isolamento de Variável
Testar um elemento por vez. Para teste de público: mantenha criativos idênticos em todos os conjuntos, mudando apenas o público.

### 3. Breakdown de Criativo Dinâmico
Acesse "Detalhamento de Elemento de Criativo Dinâmico" no Meta Ads para analisar a performance de componentes criativos individuais.

### 4. Teste A/B (Nativo)
Use o teste A/B nativo do Meta para comparar sistematicamente variáveis: criativo, público, posicionamento ou elementos personalizados.

### 5. Teste Manual
**Frase exata de Sobral:** "Eu, particularmente, não sou muito de usar essa possibilidade de teste se eu posso fazer manualmente." — Ele prefere teste manual ao automatizado quando possível.

## Framework de Sucesso em 5 Passos

1. Entender a estrutura da campanha
2. Construir campanhas usando as melhores práticas
3. Identificar variáveis de alto impacto
4. Selecionar variáveis para testar
5. Definir metodologia de teste eficiente

## Filosofia Central

**Frase exata:** "Todo investimento em tráfego pago é um aprendizado contínuo."

O sucesso deriva de "prática, análise e aperfeiçoamento constante" — não de fórmulas.
