# Superpowers

Collection of advanced development workflow skills for Claude Code.

## Sub-skills

| Skill | Purpose |
|-------|---------|
| `brainstorming` | Structured ideation and exploration |
| `writing-plans` | Create implementation plans |
| `executing-plans` | Execute plans step by step |
| `test-driven-development` | TDD workflow with anti-patterns guide |
| `requesting-code-review` | Prepare and request code reviews |
| `receiving-code-review` | Process and apply review feedback |
| `using-git-worktrees` | Parallel development with git worktrees |
| `verification-before-completion` | Pre-completion validation checklist |
| `finishing-a-development-branch` | Branch cleanup and merge workflow |

## Usage

Each sub-skill has its own `SKILL.md` with specific instructions. Activate individually as needed.
