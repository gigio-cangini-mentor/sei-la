## Story

**As a** AIOS developer,
**I want** dead code and unused modules removed from the codebase,
**so that** the project is leaner, easier to understand, and has no misleading artifacts.

## Acceptance Criteria

1. Orphaned core modules with zero consumers are either wired or removed: `ideation-engine.js`, `timeline-manager.js`
2. Phantom module references cleaned: any remaining `try/catch` requires for non-existent modules are removed
3. Unused exports identified and removed from core modules (exports that no file imports)
4. Dead code in `cloud-brain.js` identified and removed (commented-out code, unreachable branches)
5. Dead code in `server.js` identified and removed
6. All existing tests still pass after cleanup
7. `npx aios-core doctor` still passes
8. No new test failures introduced
